## How to use these examples

- Match the **register** (warmth, sentence length, vocabulary) — don't copy the exact words.
- Examples are written in American English. Keep that consistent.
- If an example violates a current brand or guardrail rule, the rule wins — flag the example as outdated.

## Adding new examples

When a great piece of copy ships, drop it into the relevant file. Over time these files become the canonical reference for "this is how Entie sounds in this format."

Conventions:

- Group by use case (e.g., hero, banner ad, push notification)
- Show 2–4 variations per use case where helpful
- Add a one-line note on *why* the line works, if it's non-obvious
